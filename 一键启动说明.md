# 图片处理工具套件 - 一键启动说明

## 快速启动

### 方式一：创建桌面快捷方式（推荐）
1. 双击运行 `创建桌面快捷方式.bat`
2. 等待提示创建完成
3. 双击桌面图标启动

### 方式二：直接运行
- 静默启动（不显示控制台）：`start_silent.pyw`
- 智能启动（自动检查依赖）：`run.py`
- 开发模式启动：`pic_tool_suite.py`

## 文件说明
- `start_silent.pyw`：静默启动脚本
- `run.py`：智能启动脚本（推荐）
- `pic_tool_suite.py`：主程序
- `创建桌面快捷方式.bat`：生成桌面快捷方式

## 日志位置
- `logs/`：日志目录
- `logs/application_YYYYMMDD_HHMMSS.log`：每次启动日志
- `logs/PicToolSuite.log`：应用主日志
- `logs/startup.log`：启动脚本日志

## 常见故障处理
- 启动失败：确认 Python 版本为 3.7+ 并已安装依赖
- 快捷方式无效：重新运行 `创建桌面快捷方式.bat`
- 功能不可用：检查日志并确认依赖是否完整

## 获取帮助时建议提供
- 错误描述
- 对应日志文件（位于 `logs/`）
- 操作系统与 Python 版本
