# 🎬 图片转GIF工具

一个功能强大的在线图片转GIF工具，支持多种自定义参数，可以将多张图片转换为动态GIF文件。

## ✨ 功能特色

- 🖼️ **多图片上传**：支持拖拽上传，一次性上传多张图片
- 🎛️ **自定义参数**：帧延迟、循环次数、图片质量、尺寸设置
- 🔄 **拖拽排序**：可以通过拖拽调整图片顺序
- 📱 **响应式设计**：支持桌面端和移动端
- 🚀 **纯前端实现**：无需服务器，完全在浏览器中运行
- 💾 **一键下载**：生成后可直接下载GIF文件

## 📁 文件说明

- `index.html` - 主要的在线版本（推荐使用）
- `index-offline.html` - 离线版本（仅动画预览，无法生成真正的GIF）
- `test.html` - 测试页面，用于验证GIF编码功能
- `start-server.py` - Python HTTP服务器启动脚本

## 🚀 使用方法

### 方法一：本地文件运行（推荐）

1. 双击运行 `start-server.py`
2. 浏览器会自动打开 `http://localhost:8000/index.html`
3. 开始使用工具

### 方法二：直接打开HTML文件

1. 直接双击 `index.html` 文件
2. 如果外部库加载失败，会自动使用内置编码器

### 方法三：使用离线版本

1. 双击 `index-offline.html` 文件
2. 可以预览动画效果，但无法生成真正的GIF文件

## 🎯 使用步骤

1. **上传图片**：点击或拖拽上传2张以上的图片
2. **调整参数**：
   - 帧延迟：控制动画播放速度（毫秒）
   - 循环次数：设置动画循环次数（0为无限循环）
   - 图片质量：选择输出质量
   - 尺寸设置：设置输出GIF的宽高
3. **排序图片**：拖拽图片调整播放顺序
4. **生成GIF**：点击"生成GIF"按钮
5. **下载文件**：生成完成后点击下载按钮

## 🔧 技术实现

- **外部库**：优先使用 `omggif` 库（不需要Web Worker）
- **备用方案**：内置简单GIF编码器
- **图片处理**：使用Canvas API处理图片
- **颜色量化**：简单的颜色量化算法
- **文件格式**：标准GIF89a格式

## 🌐 浏览器支持

- Chrome 60+
- Firefox 55+
- Safari 12+
- Edge 79+

## ⚠️ 注意事项

1. **文件大小**：建议单张图片不超过5MB
2. **图片数量**：建议不超过50张图片
3. **输出尺寸**：建议不超过800x800像素
4. **网络连接**：首次使用需要网络连接加载外部库

## 🐛 常见问题

### Q: 生成GIF时出现错误
A: 请尝试以下解决方案：
1. 刷新页面重试
2. 使用Python服务器运行（避免跨域问题）
3. 检查图片格式是否支持
4. 减小图片尺寸或数量

### Q: 生成的GIF文件很大
A: 可以尝试：
1. 降低图片质量
2. 减小输出尺寸
3. 减少帧数
4. 增加帧延迟

### Q: 无法加载外部库
A: 工具会自动使用内置编码器，功能不受影响
