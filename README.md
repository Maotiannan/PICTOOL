# 图片处理工具套件（PicTool）

一款集成批量水印、GIF 制作、图片转 PDF 等能力的桌面端工具，面向日常图片处理场景，提供统一界面与批量化流程。

## 功能概览
- 批量水印：自定义文字、水印位置、透明度、颜色、大小等
- GIF 制作：多图生成 GIF，支持帧间隔、循环次数、尺寸与质量
- 图片转 PDF：多图合并为 PDF，支持页面顺序与页面参数设置
- 统一界面：所有功能在同一应用内切换

## 环境要求
- Python 3.7+
- Windows / macOS / Linux
- 推荐内存 4GB+

## 快速开始（推荐）
1. 双击运行 `run.py`
2. 脚本会自动检查并安装依赖
3. 自动启动图形界面

## 手动安装
1. 安装 Python 3.7+
2. 安装依赖
   ```bash
   pip install -r requirements.txt
   ```
3. 启动主程序
   ```bash
   python pic_tool_suite.py
   ```

## 使用流程（通用）
1. 在左侧工具栏选择功能
2. 选择包含图片的文件夹
3. 根据需求调整参数
4. 预览效果
5. 开始处理

## 输出目录
处理后的文件默认保存在原文件夹的子目录中：
- 水印图片：`Watermarked_Images/`
- GIF 文件：`_gif/`
- PDF 文件：`_pdf/`

## 常见问题
- 启动失败：请确认 Python 版本与依赖是否安装完整
- 处理大量图片变慢：属于正常现象，可观察进度条或等待完成
- 中文字体显示异常：请确认系统已安装中文字体
- GIF 文件过大：可降低质量、缩小尺寸或增加帧间隔
