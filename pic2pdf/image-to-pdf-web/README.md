# 图片生成PDF工具 (Web版)

## 项目介绍
这是一个基于Web技术的图片转PDF工具，使用HTML、CSS和JavaScript开发。该工具允许用户通过拖放或选择的方式上传多张图片，并按照需要的顺序将它们合并成一个PDF文档。

## 功能特点
- 简洁美观的用户界面
- 支持JPG、JPEG和PNG格式图片
- 支持拖放上传和多选上传
- 实时图片预览
- 灵活调整图片顺序（上移/下移）
- 可删除不需要的图片
- 自定义PDF文件名
- 在生成的PDF文件名中自动添加日期
- 高性能处理大量图片（优化版）

## 性能优化
本版本针对大量图片处理进行了多项性能优化：
- **虚拟列表技术**：只渲染可视区域的图片，支持高效处理上千张图片
- **缩略图优化**：生成小尺寸缩略图，减少内存占用
- **Web Worker**：在后台线程处理PDF生成，避免界面卡顿
- **批量处理**：分批处理图片，避免内存溢出问题
- **图片压缩**：自动调整大图尺寸，提高PDF生成速度
- **内存管理**：定期释放不可见图片资源，减少内存占用
- **进度反馈**：实时显示处理进度，提升用户体验

## 技术栈
- HTML5
- CSS3
- JavaScript (ES6+)
- Web Worker API
- Canvas API
- jsPDF库 (用于生成PDF)

## 使用方法
1. 打开`index.html`文件在浏览器中运行该应用
2. 输入要生成的PDF文件名
3. 通过以下方式添加图片：
   - 点击"选择图片"按钮选择多个图片文件
   - 或将图片直接拖放到指定区域
4. 在预览列表中查看已添加的图片
5. 可以通过以下操作管理图片：
   - 点击图片查看大图预览
   - 使用上下箭头按钮调整图片顺序
   - 使用删除按钮移除不需要的图片
6. 设置完成后，点击"生成PDF"按钮创建PDF文件
7. 浏览器会自动下载生成的PDF文件

## 本地部署
1. 下载或克隆本项目到本地
2. 使用现代浏览器(如Chrome、Firefox、Edge等)打开`index.html`文件
3. 无需安装任何依赖或运行服务器，可直接在浏览器中使用

## 处理大量图片的建议
- 尽量使用Chrome或Edge等现代浏览器获得最佳性能
- 对于超大图片，程序会自动调整尺寸以提高性能
- 如果处理超过100张图片，建议给浏览器分配更多内存
- 在处理过程中，请勿关闭或刷新浏览器页面

## 注意事项
- 所有处理都在本地进行，图片不会上传到任何服务器
- PDF生成时间与图片数量和大小成正比
- 不同浏览器对文件操作的支持可能有所不同，推荐使用最新版Chrome浏览器

## 与原Python版本的区别
- 本Web版可直接在浏览器中运行，无需安装Python或其他依赖
- Web版界面更加现代美观
- Web版使用拖放功能，操作更加便捷
- Web版生成的PDF直接通过浏览器下载，不会保存在原图片文件夹中
- Web版支持更高效的大量图片处理，针对性能进行了多项优化
